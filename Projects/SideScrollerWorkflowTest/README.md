# SideScroller Workflow Test

## Status

Phase 1 gameplay prototype implemented for workflow validation.

## Concept

A short, single-level 3D side-scrolling jump-and-run built entirely from Unity primitives and native effects.

## Technical Baseline

- Target platform: Windows Desktop
- Unity: 6000.3.19f1
- Rendering: Built-in Render Pipeline
- External assets: None
- Additional packages: None

## Controls

- Move: A/D or Left/Right Arrow
- Jump: Space

## Scope

The prototype contains one short level, a following side camera, gaps, elevation changes, one moving platform, one horizontally moving obstacle, fall reset, landing dust, a goal particle effect, and clear success feedback. Phase 2 textures are explicitly excluded.

## Entry Point

Open `Source/SideScrollerWorkflowTest` in Unity and run `Assets/Scenes/SideScroller.unity`.
